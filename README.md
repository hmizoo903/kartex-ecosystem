# KARTEX (KTX) - Digital Asset & Settlement Simulation Protocol

KARTEX is a full-stack Web Application built to simulate a next-generation decentralized liquidity network and digital asset ecosystem.

## 🚀 Features
- **Dynamic Tokenomics & Landing Page**: Real-time stats, interactive roadmap, and whitepaper architecture.
- **Authentication System**: Secure user signup, login, and session-based access control with hashed passwords.
- **Trading & Wallet Ecosystem**: Demo USD balance funding, instant KTX buy/sell execution, and full transaction logs.
- **Interactive Price Chart**: Live price movement visualizer built with `Chart.js`.
- **Admin Control Panel**: Real-time asset pricing adjustments and user wallet balance overrides.

## 🛠️ Tech Stack
- **Backend**: Python 3 / Flask
- **Database**: SQLite / Flask-SQLAlchemy
- **Authentication**: Flask-Bcrypt
- **Frontend**: HTML5, Modern CSS Grid/Flexbox, JavaScript (ES6)
- **Charts**: Chart.js
